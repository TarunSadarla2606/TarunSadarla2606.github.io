Assign. 01

Tableau — Global Superstore Dataset

Tableau · Bar Charts · Shipping Cost Analysis

Exploratory visualization of the Global Superstore retail dataset across markets, segments, and product categories. Built side-by-side bar visualisations and multi-dimensional analytical views exploring sales, profit, and shipping metrics.

Key visualizations

  • ▸Market × Sales side-by-side bars — sum of shipping costs per sub-category across Furniture, Office Supplies, and Technology
  • ▸Key finding: Phones and Copiers have the highest shipping costs in Technology (Copiers: $159,496)
  • ▸Additional analytical views exploring sales, profit, and shipping metrics across markets

★ What this assignment taught — uniquely

Visualization as a discovery tool, not a reporting tool. The shipping cost finding (Copiers dominating Technology despite being less frequently purchased than Phones) only became visible when the data was broken down by sub-category rather than category. This experience of using visualization as an exploratory instrument — where the chart reveals something you weren't looking for — is a skill that directly applies to medical imaging data analysis, where summary statistics routinely hide subgroup patterns.

Assign. 02

D3.js + Power BI — Iris & Tech Layoffs

D3.js · Observable · Power BI · Pie Charts · Stacked Bars

Part A — D3.js via Observable (Iris dataset)

Iris dataset (150 samples, 3 species, 4 features) visualized using D3.js in an Observable notebook. Pie chart showing perfectly balanced class distribution (50 each for Setosa, Versicolor, Virginica). Additional multivariate analysis of sepal/petal dimensions using D3 interactive elements.

Observable notebook: observablehq.com/d/aa9eb9017a90976c

Part B — Power BI (Tech Layoffs)

Tech layoffs dataset analyzed in Power BI. Stacked column chart (companies × total layoffs, sorted descending). Pie chart of layoffs by company filtered to Africa, Australia, and South America — showing relative regional contribution with percentage labels.

★ What this assignment taught — uniquely

D3.js — from declarative to programmatic visualization. Tableau and Power BI abstract the chart into drag-and-drop operations. D3.js requires you to write the mapping from data to visual encoding in JavaScript — specifying exactly how data values translate to pixel positions, colors, and shapes. Working through Observable made the underlying structure of every chart visible: scales, axes, selections, and data binding. This gave me a fundamentally different mental model of what a visualization is — a function from data to visual marks — which transfers directly to designing custom clinical data visualizations that standard tools can't produce.

Final

Tableau Advanced Analytics + DAX + Figma SVG

Tableau · DAX Formula · Standard Deviation · Figma Vector Design

Advanced Tableau analytics combining computed statistical measures, custom DAX formulas, and Figma-designed SVG icons integrated into a dashboard. Demonstrates the full pipeline from data computation to visual presentation design.

Key components

  • ▸Sum and Standard Deviation of Sales Amount — aggregate statistics with std dev breakdown by Country
  • ▸Custom Quick Measure via DAX: Standard Deviation of Sales by Country using CALCULATE(STDEV.P([Sales Amount]), ALLEXCEPT(Table, Table[Country]))
  • ▸Figma SVG design: custom vector icons designed, exported as SVG, code cleaned, and integrated into Tableau dashboard

★ What this assignment taught — uniquely

DAX as a semantic layer for computed metrics. DAX (Data Analysis Expressions) is a formula language that operates on column-level calculations with context awareness — ALLEXCEPT filters work differently from what you'd expect from SQL or Python. Writing a formula that computes standard deviation per country while respecting other filter contexts made the difference between row-context and filter-context calculation concrete, which is a concept that appears in any database-backed dashboard system.

Design integration — SVG from Figma to dashboard. Designing custom icons in Figma, exporting clean SVG, and integrating them into a Tableau dashboard was the only project-adjacent experience in this portfolio involving vector design tools. Understanding SVG as a code format (not just an image format) — and being able to clean the exported code — is a transferable skill for any web-based portfolio or visualization work.
